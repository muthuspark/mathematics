cos(cos⁻¹(x)) = x, for -1 ≤ x ≤ 1

#Trignometry #Basic Inverse Relations

To prove that \(\cos(\cos^{-1}(x)) = x\) for all \( -1 \leq x \leq 1 \), we can follow these steps:

  1. \[ \cos(\theta) = x \]\[ 0 \leq \theta \leq \pi \]
  2. \[ \cos(\cos^{-1}(x)) \]\[ \cos(\cos^{-1}(x)) = \cos(\theta) = x \]

  3. Verification with Specific Values:

    • For \( x = 0.5 \), we have: \[ \cos^{-1}(0.5) = \frac{\pi}{3} \quad \text{and} \quad \cos\left(\frac{\pi}{3}\right) = 0.5 \]
    • For \( x = 1 \): \[ \cos^{-1}(1) = 0 \quad \text{and} \quad \cos(0) = 1 \]
    • For \( x = -1 \): \[ \cos^{-1}(-1) = \pi \quad \text{and} \quad \cos(\pi) = -1 \]
  4. \[ \cos(\cos^{-1}(x)) = x \]
