cos(A) + cos(B) = 2cos((A+B)/2)cos((A-B)/2)

#Trignometry #Sum-to-Product Formulas

To prove the trigonometric identity \(\cos A + \cos B = 2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)\), we can use Euler’s formula and properties of complex numbers or product-to-sum identities. Here, we will use the product-to-sum identity approach.

  1. \[ \cos A = \text{Re}(e^{iA}) \quad \text{and} \quad \cos B = \text{Re}(e^{iB}) \]
  2. \[ \cos A + \cos B = \text{Re}(e^{iA}) + \text{Re}(e^{iB}) = \text{Re}(e^{iA} + e^{iB}) \]
  3. \[ e^{iA} + e^{iB} = e^{i(A+B)/2} \left(e^{-i(A-B)/2} + e^{i(A-B)/2}\right) \]
  4. \[ e^{-i(A-B)/2} + e^{i(A-B)/2} = 2\cos\left(\frac{A - B}{2}\right) \]
  5. \[ \text{Re}(e^{iA} + e^{iB}) = \text{Re}\left(e^{i(A+B)/2} \cdot 2\cos\left(\frac{A - B}{2}\right)\right) = 2\cos\left(\frac{A + B}{2}\right)\cos\left(\frac{A - B}{2}\right) \]

Thus, we have shown that:

\[ \cos A + \cos B = 2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right) \]

The identity is proven. The final answer is:

\[ \boxed{\cos A + \cos B = 2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)} \]
